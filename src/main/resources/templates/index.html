Questão 1


Acerca de sistemas de desenvolvimento web, assinale a opção correta.

A) Servlet é uma classe do Java que possibilita ampliar os recursos de servidores web, desenvolvida para permitir conteúdos dinâmicos orientados ao usuário.

B) Para utilizar bancos de dados relacionais em aplicações desenvolvidas em JSP, é obrigatória a utilização do Hibernate, que é um framework que realiza o mapeamento objeto/relacional.

Justificativa

A utilização do Hibernate não é obrigatória na camada de persistência, podemos utilizar acesso JDBC direto.

C) O Ajax permite interagir com dados textuais nos formatos UTF-8 e XML, porém restringe o acesso a JSON (Java Script Object Notation) e a bancos de dados relacionais.

Justificativa

É possível utilizar o protocolo SOAP com o Ajax e não é restrito a bancos relacionais (vide integração com Mongo)

D) No XMLHttpRequest, utilizado para trocar dados com um servidor, com o intuito de melhorar sua usabilidade, o método open () aceita somente requisições no modo asynchronous.

Justificativa

Existe uma sobrecarga do método open onde é possível passar o tipo de comportamento através de um boolean como parâmetro.

E) O JSF (Java Server Faces) permite usar tags customizadas limitadas a páginas JSP, com vistas a encapsular a segurança na forma nativa do acesso aos JavaBeans.

Justificativa

A utilização de tags customizadas não se limita a páginas JSP, podemos usar as tags em arquivos XHTML.

Questão 3


São apenas tipos de componentes executados em servidores Web:

A) Beans, Servlets e J2EE.

Justificativa

J2EE não é um componente, é um conjunto de especificações com o objetivo de facilitar o desenvolvimento de softwares utilizando a linguagem Java.

B) JVM, Servlets e JSP.

Justificativa

JVM (Java Virtual Machine) não é um componente executado em servidor web. A JVM é basicamente uma ponte entre a linguagem Java e o Sistema Operacional, permitindo então que softwares Java rodem em qualquer SO.

C) Beans, Servlets e JSP.

D) Beans, Swing e JSP.

Justificativa

Swing não é um componente executado em servidor web, sua aplicação é voltada para desenvolvimento Desktop (JSE).

E) Beans, Swing e JVM.

Justificativa

Swing não é um componente executado em servidor web, sua aplicação é voltada para desenvolvimento Desktop (JSE).

Questão 4


Analise os itens a seguir sobre JEE e EJB.

I. Um servidor J2EE fornece contêineres EJB e Web.

II. O contêiner EJB gerencia a execução de EJBs em aplicações J2EE.

III. O contêiner Web gerencia a execução de páginas JSP e componentes servlet em aplicações J2EE.

IV. Um session bean representa um único cliente dentro do servidor J2EE. Para acessar um aplicativo que é instalado no servidor, o cliente invoca os métodos do session bean.

A) I, II, III e IV.

B) I e II, apenas.

Questão 5


Spring Framework é uma plataforma Java completa que fornece suporte de infraestrutura para o desenvolvimento de aplicações Java. Acerca das características do framework Spring 3.0, assinale a opção correta.

A) Na arquitetura Spring MVC Web, o Validator é uma classe opcional que pode ser invocada para validar dados de formulários.

Justificativa

Sim, podemos criar classes implementando a Interface Validator do pacote org.springframework.validation para validar os dados de um Modelo recebidos pelo Controller que foram enviados por um formulário em uma JSP.

B) A injeção de dependência é feita após a criação do objeto, por meio dos métodos set de uma classe no estilo JavaBean, e não no momento da criação do objeto, tendo-se em vista que passar muitos argumentos no construtor pode tornar-se dispendioso.

Justificativa

A injeção de dependência do Spring é realizada quando o Spring fornece o objeto já instanciado para o Construtor da Classe que utilizará o objeto, ou diretamente em uma variável de classe utilizando a anotação @Autowired.

C) A interface BeanFactory gerencia beans definidos em arquivos XML e trata recursos de mensagens.

Justificativa

A interface BeanFactory não trata recursos de mensagens, apenas a ApplicationContext (que faz tudo que a BeanFactory faz), tem essa possibilidade.

D) O controlador AbstractWizardFormController, do módulo Spring MVC, permite suporte para o preenchimento de formulários a partir de determinada solicitação.

Justificativa

Apesar de depreciado, o papel do AbstractWizardFormController é gerenciar a criação de um fluxo de formulário (Wizard) e não dar suporte para preenchimento de formulários.

E) A porta de entrada do navegador web para a arquitetura Spring MVC Web é a componente Interface (JSP/HTML).

Justificativa

O View do MVC do Spring é baseado em JSP. O Spring só aceita a extensão HTML com a utilização do Thymeleaf com o Spring Boot.
