Questão 9 Vamos lá!

Considerando a estrutura de uma árvore binária:


public class BinaryTree {
   int valor;
   BinaryTree left;
   BinaryTree right;
}

Desenvolva um método que dado um nó da árvore calcule a soma de todos os nós subsequentes.

public int somarValorDosNos(BinaryTree atual) {
   int soma = atual.getValor();
   soma += atual.getLeft() != null ? somarValorDosNos(atual.getLeft()) : 0;
   soma += atual.getRight() != null ? somarValorDosNos(atual.getRight()) : 0;
   return soma;
}
